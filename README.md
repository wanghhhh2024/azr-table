# azr-table

`azr-table` 是一个基于 Vue 和 Element Plus 的可编辑表格组件，并包含一些类似 Excel 的便捷操作

## 功能点

* 区域选取
   - 拖拽选中
   - 鼠标拖动自动填充     
* 复制/粘贴
* 单元格编辑
   - 文本框
   - 下拉选择
   - 级联选择   
* 右键
  - 向上/向下新增行
  - 右键复制/粘贴
  - 删除行/区域选中的行

## 安装运行

```bash
# 安装依赖
pnpm install

# 如果 pnpm install 安装失败，尝试使用下面的命令安装依赖
pnpm install --ignore-scripts

# 本地开发环境启动
pnpm dev

# 生产环境打包
pnpm build
```
